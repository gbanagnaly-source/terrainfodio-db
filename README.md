# FODIO TERRAIN - Google Sheets Integration

Application de gestion de terrains avec synchronisation Google Sheets.

## Déploiement sur Vercel

1. Allez sur [vercel.com](https://vercel.com)
2. Importez ce dossier
3. Cliquez sur "Deploy"

## Configuration

L'application se connecte automatiquement à Google Sheets via l'API configurée.

## Technologies

- Next.js 14
- React 18
- Tailwind CSS 3
- shadcn/ui
- Google Sheets API
